## 설계 원칙

- **최근 표결만(2025년 이후)** — 현재 의원 구성이 실제로 던진 표라 "내 입장 vs 표결" 연결이 깔끔하고, 사용자에게 지금의 이슈로 와닿음.
- **정치 용어 0** — 생활 언어 상황 질문. 법안명·전문용어는 ▼출처로만.
- **Ⓐ/Ⓑ 장단점 비교형** — 양쪽 모두 👍장점+👎단점 대칭. 정치 지식 없이 끌리는 쪽 선택.
- **변별력** — 전 문항 *정당이 실제 갈린* 표결에 앵커링.
- **채점 규약**: `option_a` = 앵커 법안 **찬성** 방향, `option_b` = **반대** 방향.
- 7축 고정 대신 **최근 정국 이슈 중심**(쟁점에 '교육' 추가). 안보·외교/환경·기후/젠더·가족은 2025년 이후 *정당이 갈린* 표결이 없어 이번 세트에서 빠짐.

## 4차까지의 변화

1차(14, 만장일치 다수) → 2차(11, 변별력) → 3차(장단점 Ⓐ/Ⓑ 접근성) → **4차(최근 표결만 2025+)**.
직전 세트의 2024년 앵커(후쿠시마·양곡·상속세·한미방위·농어업재해·방송법)는 최근성 기준으로 제외.

## 교차검토 포인트

- 8번(집회)·1번(법인세)은 정치적으로 민감 → 톤 외부 검수.
- 6·7번 교육은 일상적이라 접근성 강점이나, 혁신·진보가 학생인권 관점에서 갈리는 비대칭 → 프레이밍 검수.
- Ⓐ/Ⓑ **표시 순서 무작위화** 권장(현재 Ⓐ=찬성방향 고정 → UI 셔플, 내부 매핑 유지).
- 안보·외교/환경·기후/젠더·가족 **공백** — 최근 변별 표결 생기면 추가하거나, 계류 법안으로 보완할지 결정 필요.
